Detailed Description

This section specifies callback function of the driver. The callback function is called by the driver to notify software interrupt. The interrupt flag is cleared by the driver.

Typedef Documentation

typedef void(* LLWU_CALLBACK)(uint32 flags)

LLWU_CALLBACK function declaration.

This callback is called by the LLWU module upon recovery from VLLS modes. The service routine shall restore I/O and release latch states by PMC_SetACKISO() macro.
