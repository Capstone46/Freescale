#define DMA_SINGLE_ERQ_IRQ_CONFIG (   bcnt,
  ssize,
  dsize 
)

DMA transfer long word from source destination. DMA channel 1 request is generated by minor loop done of DMA channel 0. IRQ is enabled.

Parameters
bcnt- number of bytes to be transfered in block (up to 0xFFFFF)
ssize- source transfer size (DMA Data Transfer Sizes, see in dma.h)
dsize- destination transfer size (DMA Data Transfer Sizes, see in dma.h)
Note
Implemented as function call.
