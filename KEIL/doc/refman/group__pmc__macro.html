Macro Definition Documentation

#define PMC_GetACKISO ( )

Macro returns state indicating whether certain peripherals and the I/O pads are in a latched state as a result of having been in a VLLS mode. Writing one to this bit when it is set releases the I/O pads and certain peripherals to their normal run mode state - see PMC_SetACKISO().

Returns
TRUE - Certain peripherals and I/O pads are in an isolated and latched state
FALSE - Peripherals and I/O pads are in normal run state
Note
Implemented as inlined macro.
#define PMC_GetLVDF ( )

Macro returns LVDF status.

Returns

FALSE - Normal operation.

#define PMC_GetREGONS ( )

Macro returns regulator status.

TRUE - Regulator is in stop regulation or in transition to/from it.
FALSE - Regulator is in run regulation.
Note
Implemented as inlined macro.
