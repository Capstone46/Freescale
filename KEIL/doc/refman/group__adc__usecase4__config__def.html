Detailed Description

Data Structure Documentation

struct sADC_MEASUREMENT

Definition of structure of 4 measured elements.

This structure is defined just for 4 ADC sequential measuremnts. Each element is long type because of limitation of DMA circular buffer (min. 16 bytes which represents 4 longs).

#define ADC_FAST_MUXB_HWTRG_XREF_DMA_CONFIG (   nbit)

Selects n-bit fast mode with HW trig. and ext. reference.

  • A power config
  • Input clock is BUSCLK
  • Clock divider set to 2 (ADCLK = BUSCLK/2, considered as 12MHz max. ADCCLK)
  • fast = short sample time, high speed sequence, shortest sample time
  • HW trigger
  • ADxxb channels selected
  • external voltage reference (VREH and VREFL)
  • one conversion
  • DMA enabled
    Parameters
    nbit- number of bits to be converted (ADC mode option (number of converted bits))
