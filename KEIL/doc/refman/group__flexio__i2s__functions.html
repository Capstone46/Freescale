FLEXIOI2S module initialization function.

Parameters
shifterOutIx- Index of shifter for transmit
shifterInIx- Index of shifter for receive
timerBclkIx- Index of timer for I2S bit clock a shift registers clk
timerFsyncIx- Index of timer for I2S FS (WS) Word Select signal
timerMclkIx- Index of timer for I2S MCLK (Master Clock)
pinDoutIx- Index of FlexIO Pin for I2S DOUT signal
pinDinIx- Index of FlexIO Pin for I2S DIN signal
pinBclkIx- Index of FlexIO Pin for I2S BCLK signal
pinFsyncIx- Index of FlexIO Pin for I2S FS signal
pinMclkIx- Index of FlexIO Pin for I2S MCLK signal
wordSize- size of the audio samples in bits (16,32)
sampleRate- Requested audio samplerate
iisMclk- I2S master clock frequency [Hz]
FlexIoClk- Clock frequency of FlexIO peripheral [Hz]
Returns
Result of Init Operation
Note
Implemented as function call.
