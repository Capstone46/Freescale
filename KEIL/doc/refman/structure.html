CREATING NEW PROJECT

Execute make_project.exe script from the "template" subfolder to create a new project. The script clones new project based on the project template.

  • After executing you can select from compilers/development tools:
    • IAR(ICC)
    • KEIL (ARMCC)
  • You can also select from debugger probes
    • j-LINK/J-Trace
    • PE micro,
    • I-Jet/JTAGjet (or U-Link Pro)
  • at the end project name should be entered.

After project name entered project based files (IAR/KEIL) will be placed into the folder named by entered project name at:
code\build\iar\projects\project name
The application source and configuration files can be found at:
code\src\projects\project name

PROJECT STRUCTURE

The project has the following directory structure:

  • [trunk] - Main installation directory
    • [build] - Project subfolder
      • [iar_] - IAR EWARM 6.7x Project Folder
        • [projects] - specified projects folder
          • [adc0_usecase] - ADC usecase project example
    • [refman] - Reference manual subfolder
      • [drivers] - Drivers documentation
    • [valreport] - Validation report manual subfolder
      • [drivers] - Drivers documentation
    • [src] - Source code repository
      • [common] - Common source code
      • [cpu] - basic cpu source code
        • [headers] - header files included
        • [IAR] - IAR based start up (asm)
        • [KEIL] - KEIL based start up (asm)
      • [devices] - external devices drivers
        • [fxio8700] - acceleromenter&magnetometer driver
        • [mma845x] - accelerometer driver
      • [drivers] - Drivers source code
        • [adc] - ADC driver code
        • [bme] - BME driver code
        • [cmp] - CMP driver code
        • [dac] - DAC driver code
        • [dma] - DMA driver code
        • [flexio] - FLEXIO driver code
          • [flexio_uart] - FLEXIO UART driver code
          • [flexio_i2c] - FLEXIO I2C driver code
          • [flexio_i2s] - FLEXIO I2S driver code
          • [flexio_spi] - FLEXIO SPI driver code
        • [ftfa] - FTFA driver code
        • [gpio] - GPIO driver code
        • [i2c] - I2C driver code
        • [llwu] - LLWU driver code
        • [lptrm] - LPTMR driver code
        • [lpuart] - LPUART driver code
        • [mcg] - MCG driver code (KLx6 only)
        • [mcg_lite] - MCG lite driver code (KLx3 only)
        • [mcm] - MCM driver code
        • [osc] - OSC driver code
        • [pit] - PIT driver code
        • [pmc] - PMC driver code
        • [rcm] - RCM driver code
        • [rtc] - RTC driver code
        • [sai] - SAI/I2S driver code
        • [sim] - SIM driver code
        • [slcd] - SLCD driver code
        • [smc] - SMC driver code
        • [spi] - SPI driver code
        • [tpm] - TPM driver code
        • [tsi] - TSI driver code (KLx6 only)
        • [uart] - UART driver code
        • [vref] - VREF driver code (KLx3 only)
      • [linker files] - linker base folder
        • [IAR] - IAR based linker file (.icf)
        • [KEIL] - KEIL based linker file (.scf)
      • [projects] - Project examples source code
        • [adc_usecase1] - ADC usecase 1 source files
      • [linker files] - linker base folder
        • [fraclib] - frac math lib
        • [freemaster] - freemmaster tool source files
        • [tss] - tss tool source files
    • [template] - New project template
      • make_project.exe - Run this script and enter input arguments to create new project
      • make_val_project.exe - Run this script and enter input arguments to create new validation project

The source code for drivers and examples are located in the drivers and projects subfolders. The project files are placed in a subfolder for the respective compiler of the build folder.
