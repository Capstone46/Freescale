#define DMA_SINC_256BBUFF_SINGLE_ERQ_IRQ_CONFIG (   bcnt,
  ssize,
  dsize 
)

DMA transfer from source circular buffer (256 words) to one destination.

Parameters
bcnt- number of bytes to be transfered in block (up to 0xFFFFF)
ssize- source transfer size (DMA Data Transfer Sizes, see in dma.h)
dsize- destination transfer size (DMA Data Transfer Sizes, see in dma.h)
Note
Implemented as function call.
#define MCU_MKL43Z256

Define processor device.

Supported devices - MCU_MKL43Z256 (MCU_MKL46Z256 temporarily used)

#define TPM_MODULE_SYSCLK_DIVBY1_EDGEALIGN_DMA_PITSTART_OVSTOP_CONFIG

Edge aligned timer mode with source clock divided by 1.

Timer has input trigger from PIT CH0. PIT will trigger TPM0 to start counting until overflow, then stopped and waits for trigger again.
