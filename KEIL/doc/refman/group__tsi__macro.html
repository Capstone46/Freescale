TSI used chanels calibration function. Function performs calibration of selected TSI channels and updates calibration treshhold values.

Parameters
module- TSI0
cfg- TSI Configuration Structures
achan- pointer to channels array
asize- size of channels array
sens- TSI Touch Sensitivity
*pCALIB- pointer to tCALIB data structure.
Returns
0 - SUCCESS
> 1 - ERROR - channel number+1 which caused an error of calibration
Note
Implemented as function call.
