Detailed Description

This section specifies default configuration structures. Note that application specific configuration structure shall be created by the user and stored in the appconfig.h file. The easiest way to create user specific configuration structure is to copy default structure into appconfig.h file, rename it and update peripheral behavior by updating SET() and CLR() macros on each register field.

#define PLL_MODULE_CONFIG_PBE_48MHZ_OSC_8MHZ

PLL bypass external. PLL with very high frequency range, high gain. External divider set to 4 (8MHz/4 = 2MHz) and resulting input PLL frequency set to 2MHz (considering 8MHz external crystal). Multiplication factor set to 24 (24*2MHZ = 48MHz). MCGOUTCLK is derived from externall clock, it means 8MHz in this case.

Note
Please, follow the MCG mode moving restrictions provided in RM - MCG mode state diagram.
Warning
Please, follow the harware restriction related to RANGE and HGO in order to initialyye PLL and external reference properly. Especially feedback resitor and crystal frequency range.
