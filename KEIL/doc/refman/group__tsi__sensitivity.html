#define TSI_VERY_LOW_SENSITIVITY
 use 100% extra treshhold range e.g. calculated THRESH = 500, TRESL = 450 -> use THRESH = 1000, TRESL = 0

 use 12.5% extra treshhold range
 
#define TSI_VERY_HIGH_SENSITIVITY
 use 6.25% extra treshhold range
 
#define TSI_ULTRA_HIGH_SENSITIVITY
 use 0% extra treshhold range
 

Detailed Description

These macros specifies sensitivity factors used in TSI_Calib () function to more precisely configure treshhold values (considering HW - touch electrodes design).
