#define ADC_FAST_HWTRG_XREF_MUXB_CONFIG (   nbit)

Selects n-bit fast mode with HW trig. and ext. reference.

  • Normal power config (MUXb)
  • Input clock is BUSCLK
  • Clock divider set to 2 (ADCLK = BUSCLK/2, considered as 12MHz max. ADCCLK)
  • fast = short sample time, high speed sequence, shortest sample time
  • HW trigger
  • ADxxb channels selected
  • external voltage reference (VREH and VREFL)
  • one conversion
    Parameters
    nbit- number of bits to be converted (ADC mode option (number of converted bits))
#define HSCMP_SAMPLED_INTDRV_FILTER_MODE_IER_OUTEN_CONFIG (   flt_count,
  samp_per 
)

Comparator setting:

  • sampled filtered mode internally driven
  • filter sample count set by parameter flt_count
  • sampling disabled
  • rising edge interrupt enabled
  • filter disabled
  • CMP output pin enabled
    Parameters
    flt_countfilter sample counts select from 2 to 7
    samp_perselect one sample periode 1 to 255
    Note
    If flt_count is set to 1 then sampled non-filtered mode is configured.

#define TPM_MODULE_SYSCLK_DIVBY1_EDGEALIGN_INTRG_CROT_CONFIG

Edge aligned timer mode with source clock divided by 1.

Timer has input trigger from CMP0 output. The CMP output rising will re-load the TPM (initiate PWM output to their default sates).
