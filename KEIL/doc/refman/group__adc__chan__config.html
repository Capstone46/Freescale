Detailed Description

This section specifies default configuration structures for ADC channel (result & status and control register). The 16-bit SAR ADC module device contains 4 result registers. Note that application specific configuration structure shall be created by the user and stored in the appconfig.h file. The easiest way to create user specific configuration structure is to copy default structure into appconfig.h file, rename it and update peripheral behavior by updating SET() and CLR() macros on each register field.
