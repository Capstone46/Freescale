FLEXIO_I2C module Combo (Write/Read sequence) data function.

Parameters
address- slave chip address
pBufferOut- pointer to data buffer for write sequence
lenOut- length of data for write sequence (restricted to 13 bytes, due to limitation of FlexIO peripheral)
pBufferIn- pointer to data buffer for read sequence
lenIn- length of data for read sequence (restricted to 13 bytes, due to limitation of FlexIO peripheral)
Returns
Result of Combo (Write/Read sequence) Operation
Note
Implemented as function call.

Parameters
shifterOutIx- Index of shifter for transmit
shifterInIx- Index of shifter for receive and ACK
timerOutIx- Index of timer for transmit
timerInIx- Index of timer for clocking of shifter registers
pinSclIx- Index of FlexIO Pin for SCL signal
pinSdaIx- Index of FlexIO Pin for SDA signal
baudRate- Requested baudrate
FlexIoClk- Clock frequency of LexIO peripheral
Returns
Result of Init Operation
Note
Implemented as function call.
