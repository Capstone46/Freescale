#define ADC_CLK6MHZ_16B_SWTRG_XREF_CONFIG

Selects 16-bit software triggered mode with external voltage reference.

  • Normal power config (MUXa)
  • Input clock is BUSCLK
  • Clock divider set to 1 (ADCLK = BUSCLK/4 = 6MHz)
  • long sample time
  • 16-bit conversion mode
  • SW trigger
  • ADxxa channels selected
  • external reference
  • continuous conversion disabled
  • hardware avegaring disabled
