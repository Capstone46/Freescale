#define LPTMR_GetCmpFlag ( )

Macro returns timer compare flag.

This macro returns timer compare flag when timer increments.

Returns
TRUE - value of counter is equal to compare value. FALSE- value of counter is not equal to compare value.
Note
Implemented as inlined macro.
#define LPTMR_Init (   cfg,
  cmpval,
  ip,
  callback 
)

LPTMR initialization function.

This function initalizes Low-Power Timer (LPTRM) block. It also installs callback function for LPTMR interrupt vector 46.

Parameters
cfgUse one of following configuration structures LPTMR Configuration Structures
cmpval16-bit integer compare register value (defines period of the interrupt generation).
ipInterrupt priority 0..3
callbackpointer to LPTMR_CALLBACK function
Note
Implemented as function call.
